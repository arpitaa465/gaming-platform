🔄 How Cookies Work in This Website

  1. User logs in with email and password
  2. Server creates a session cookie with user ID, username, and role
  3. Cookie is sent to browser and stored
  4. Every subsequent request automatically includes this cookie
  5. Server reads the cookie to identify the user
  6. When user logs out, the cookie is destroyed
